Tiagodesktop

Aumente seu desempenho, limite seus usuários no Windows

View project on GitHub

O que é o Tiagodesktop?

É uma barra de tarefas personalizável que substitui a atual barra de tarefas do Windows, a fim de focar o usuário a utilizar apenas o que foi lhe permitido. Ainda estamos aperfeiçoando o software, mas já está utilizável na versão atual.

Menu

Atualmente é possível adicionar 4 botões de uso liberado e 4 botões de uso administrativo, onde apenas o Administrador do sistema conseguiria abrir os arquivos. Para configurá-los, basta clicar no botão de configuração e em seguida selecionar qual dos botões deseja adicionar/alterar/apagar.

Limitações Extras

Para aperfeiçoar a barra, contamos com a utilização de um software em específico que reconfigura a função das teclas no windows. O programa em questão se chama SharpKeys. Normalmente reconfiguramos a tecla do Windows e a CTRL (esquerdo e direito), retirando quaisquer funções existentes nas mesmas.

Programas Extras

Segue na janela de configuração dois programas de acesso remoto, o TeamViewer e o Ammyy Admin.

Suporte ou Contato
